# Cordova Tutorial

https://www.tutorialspoint.com/cordova/index.htm

## Set up

### Requirements

- Android SDK
- Xcode

### Install Cordova

```
$ npm install -g cordova
```

### Install Plugins

```
$ cordova platform add android
$ cordova platform add ios
```

## Build

- android

```
$ cordova build android
```

- ios

```
$ cordova build ios
```

## Run

### Emulator

- android

```
$ cordova emulate android
```
- ios

```
$ cordova emulate ios
```
